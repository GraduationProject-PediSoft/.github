# Pedisoft

Degree project to acquire the title of systems engineering in the Pontificia Universidad Javeriana

# Technologies
- Angular
- Springboot
- SpringCloud
- Graphql
- Oauth2

# Architecture
The system is designed as a microservice system, using apigateway and discovery service patterns
